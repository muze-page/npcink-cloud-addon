# Image Prompts

## Shared Series Identity

Use a consistent Magick AI plugin-family visual language across future assets:

- Modern open-source developer tooling.
- Clean geometric shapes.
- Deep charcoal or clean off-white base.
- WordPress-like blue accents.
- Green validation accents.
- A small magenta/magic accent used sparingly as the Magick AI family signature.
- No official WordPress logo, MCP logo, AI company logos, or trademarked marks.

## Icon Prompt

```text
Create a square WordPress.org plugin icon for "Magick AI Cloud Addon", one plugin in the Magick AI plugin family.

This specific plugin represents a thin Cloud service connector for WordPress: Cloud Base URL and Cloud API Key settings, server-side request signing, health checks, and entitlement summaries for Magick AI hosted runtime access. It is not the ability content package, not the governance core, not the OpenClaw adapter, not a workflow engine, not a billing truth source, and not a WordPress write executor.

Visual metaphor: a compact secure cloud link between a local WordPress control-plane node and a hosted runtime service node, with a signed key/checkmark, health pulse, and entitlement badge. The cloud should look like a bounded service endpoint, not a large control dashboard.

Series identity: modern open-source developer tooling, clean geometric shapes, consistent Magick AI family style, restrained technical design.

Color direction: deep charcoal or ink base, WordPress-like blue accents, green verified/healthy accents, small magenta/magic accent used sparingly as the Magick AI family signature.

Style: flat vector, crisp, high contrast, professional, readable at 128x128 and 256x256.

Avoid: robot faces, chat bubbles, cloud control dashboards, billing dashboards, workflow timelines, approval panels, ability catalog nodes as the main subject, official WordPress logo, MCP logo, AI company logos, tiny unreadable text, fantasy magic wand imagery.
```

## Banner Prompt

```text
Create a WordPress.org plugin banner for "Magick AI Cloud Addon", size 1544x500, with a layout that also works when scaled down to 772x250.

This plugin is part of the Magick AI plugin family. Its role is the thin Cloud service connector: it stores Cloud Base URL and Cloud API Key settings, parses Cloud-issued keys, signs hosted runtime requests, probes health, and reads entitlement summaries for local display.

It complements the other Magick AI plugins:
- Magick AI Abilities handles ability definitions and ability callbacks.
- Magick AI Core handles governance, approval, preflight, and audit.
- Magick AI Adapter handles OpenClaw channel adaptation and calls Core plus the Abilities API.
This banner should clearly represent Cloud connection and signing, not Abilities, Core, or Adapter.

Visual concept: a local WordPress site control-plane node connected through a narrow secure signed link to a bounded hosted runtime cloud endpoint. Show health verification, entitlement summary, trace/signature lines, and service connection status. Do not show Cloud as the control plane.

Text: "Magick AI Cloud Addon"
Optional subtitle: "Cloud connection, signing, health checks, and entitlement summaries"

Tone: secure service connection, hosted runtime access, local control preserved, calm and professional developer tooling.

Series identity: use a visual language that can extend to Abilities, Core, and Adapter: same geometry, same base palette, different symbolic focus.

Color direction: deep charcoal or clean off-white base, WordPress-like blue, green verified/healthy accents, small magenta/magic accent used sparingly.

Composition: strong readable title area, abstract secure cloud link on the opposite side, generous margins, no critical details near edges, balanced enough for LTR and RTL contexts.

Avoid: robots, chatbot UI, cloud dashboards, billing dashboards, workflow engine graphics, approval/audit panels as the primary metaphor, ability catalog nodes as the primary metaphor, official WordPress logo, MCP logo, AI company logos, fantasy magic effects, marketing hype, small unreadable code text.
```
